This library is only built for personal practice. The source codes are pretty simple, and it's not 
very hard to learn.

- [React](https://react.dev/) to build UI.
- [Vite](https://vitejs.dev/) to build the project.
- [Vitest](https://vitest.dev/) to write test cases for components.
- [Storybook](https://storybook.js.org/docs) to automatically generate this doc.
